POST Inscripcion
Agrega una inscripcion a la base de datos
https://backend-inscripciones.herokuapp.com/api/inscripcion/

El formato para mandar los dato es este

{
    "matricula": 1953829,
    "idEntrenador": 3,
    "idDeporte": 4,
    "idAdministrador": 1946218
}
